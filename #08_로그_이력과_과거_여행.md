## 여러 커밋과 로그 이력

### 명령 log
$ git log : 기본적으로 가장 최근의 커밋부터 표시<br>
옵션 : --graph, -reverse, --all, -n<br>
<br>

## 과거로의 시간여행

## 명령 checkout
현재 브랜치에서 과거 커밋 HEAD~로 이동<br>
* 현재 상태가 깨끗(clean)해야 가능<br>
* 떨어진 HEAD 상태 - detached HEAD<br>
<br>

## detached HEAD
명령 git status에서 떨어진 HEAD 상태를 확인<br>
<br>
